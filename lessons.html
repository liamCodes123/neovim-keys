Basic Movement

h j k l

h = left
j = down
k = up
l = right

Editing

  • i → Insert mode
  • esc → Leave insert mode
  • dd → Delete line
  • yy → Copy line
  • p → Paste
  • u → Undo

LazyVim Shortcuts

  • space + e → File explorer
  • space + ff → Find files
  • space + fg → Live grep
  • shift + h/l → Switch buffers
  • space + bd → Close buffer
