Vim Movement Trainer

Reach the glowing square using h j k l.

Level: 1

Last Command: none

Controls

  • h → move left
  • j → move down
  • k → move up
  • l → move right
  • dd → destroy a wall
  • yy → duplicate player clone
  • u → undo clone
